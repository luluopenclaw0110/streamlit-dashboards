### Tab 切換邏輯

```python
# Session State 設計
if 'current_tab' not in st.session_state:
    st.session_state.current_tab = '主頁'

tab1, tab2 = st.tabs(['📊 主頁', '🔍 個股查詢'])

with tab1:
    # 主頁：只顯示市場情緒 + 宏觀指標 + 預設股票
    render_market_mood()
    render_macro_indicators()
    render_default_chart()  # 預設 2330

with tab2:
    # 個股查詢：懒加载
    selected = st.text_input('輸入股票代號或名稱', placeholder='例如：2330 或 台積電')
    if selected:
        render_full_chart(selected)  # 點擊查詢才載入
```

## ⚡ 四、懒加载實作策略

### 方案 A：按需查詢（推薦）
```python
@st.cache_data(ttl=300)
def lazy_load_stock(ticker):
    """只有在调用时才查询，不预先加载"""
    info = yf.Ticker(ticker).info
    return info
```

### 方案 B：背景緩存
```python
# 個股查詢頁 - 使用 st.fragment 局部刷新
@st.fragment
def render_stock_query():
    ticker = st.text_input('股票代號', key='query_ticker')
    if ticker:
        with st.spinner('載入中...'):
            df = get_yf_history(ticker)  # 只在有輸入時才查
            st.plotly_chart(df)
```

### 懒加载時機
| 情境 | 行為 |
|------|------|
| 進入「個股查詢」Tab | ❌ 不自動查詢 |
| 輸入股票代碼 | ✅ 按下查詢鈕才載入 |
| 切換時間範圍 | ✅ 局部刷新 K線圖 |
| 切換技術指標 | ✅ 局部刷新，不重查報價 |
